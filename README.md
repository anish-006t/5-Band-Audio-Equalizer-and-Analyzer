# 5-Band Audio Equalizer and Spectrum Analyzer

A real-time analog audio processing system built using active Butterworth filters, op-amps, and analog signal processing techniques. The project consists of a 5-band analog audio equalizer for frequency-selective audio enhancement and an audio spectrum analyzer with LED visualization. 

---

## Features

- 5-band real-time audio spectrum analysis
- 5-band analog audio equalizer
- 4th-order Butterworth active filters
- Multiple Feedback (MFB) filter topology
- LED-based amplitude visualization
- LTSpice simulation and frequency response validation

---

## Technologies Used

- LTSpice
- NE5532 Op-Amps
- Analog Signal Processing
- Active Filters
- Butterworth Filter Design
- Frequency Response Analysis

---



## Simulation

The filters were designed and verified in LTSpice using AC sweep and Bode plot analysis to validate:

- Cutoff frequencies
- Gain response
- Butterworth characteristics
- Frequency band separation
